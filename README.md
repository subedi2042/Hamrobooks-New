# Hamrobooks New

Hamrobooks is a modern e-commerce platform for book enthusiasts, built with a robust and scalable technology stack. This project aims to provide a seamless shopping experience for discovering, purchasing, and managing books.

Hamrobooks is designed to be more than just an online bookstore. It is a community-driven platform that supports authors and readers alike, featuring a rich product catalog, secure checkout, and future plans for community memberships and mobile applications.

## üõ† Technology Stack

### Frontend & Core

- **Framework**: [Next.js](https://nextjs.org/) (React)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **State Management**: React Context API
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)

- **Database**: [Cloud Firestore](https://firebase.google.com/docs/firestore) (Firebase)
- **Deployment**: [Vercel](https://vercel.com/)

## üõ† Getting Started

### Prerequisites

- Node.js (Latest LTS)
- npm or yarn

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/subedi2042/Hamrobooks-New.git
   ```

2. Navigate to the web directory:

   ```bash
   cd web
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Run the development server:

   ```bash
   npm run dev
   ```
